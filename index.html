User Management
  • GET /users

    Method: GET

    Endpoint: /users

    Description: Retrieve a list of users.

    Example Request:

    GET https://api.example.com/users

    Example Response:

    {
  "users": [
    { "id": 1, "name": "John Doe" },
    { "id": 2, "name": "Jane Smith" }
  ]
}
  • POST /users

    Method: POST

    Endpoint: /users

    Description: Create a new user.

    Example Request:

    POST https://api.example.com/users
{
  "name": "New User",
  "email": "newuser@example.com"
}

    Example Response:

    {
  "id": 3,
  "name": "New User",
  "email": "newuser@example.com"
}
Product Management
  • GET /products

    Method: GET

    Endpoint: /products

    Description: Retrieve a list of products.

    Example Request:

    GET https://api.example.com/products

    Example Response:

    {
  "products": [
    { "id": 1, "name": "Product A" },
    { "id": 2, "name": "Product B" }
  ]
}
  • POST /products

    Method: POST

    Endpoint: /products

    Description: Create a new product.

    Example Request:

    POST https://api.example.com/products
{
  "name": "New Product",
  "price": 99.99
}

    Example Response:

    {
  "id": 3,
  "name": "New Product",
  "price": 99.99
}
