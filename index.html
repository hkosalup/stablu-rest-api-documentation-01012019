Stablu SIBMA Proof of Concept API Documentation

This is the REST APIs

Organizations all company we call organizations
  • POST/organizationscreate an organization

    Method: GET

    Endpoint: /organizations

    Description: Adds an organization to the system by Stablu

    Example Request:

    GET https://api.example.com/users

    Parameters:

                        {
                      "users": [
                        { "id": 1, "name": "John Doe" },
                        { "id": 2, "name": "Jane Smith" }
                      ]
                    }

    Responses:

                        {
                      "users": [
                        { "id": 1, "name": "John Doe" },
                        { "id": 2, "name": "Jane Smith" }
                      ]
                    }
