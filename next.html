POST /organizations

Adds an organization to the system.

Description

Adds an organization to the system by its attributes.

Parameters

body
object
Organization attributes to add
{
  "id": "d290f1ee-6c54-4b01-90e6-d701748f0851",
  "name": "Stablu Company",
  "description": "Stablu is a tech innovation company",
  "logo": "https://stablu.net/logo.png",
  "created_at": "2016-08-29T09:12:33.001Z",
  "updated_at": "2016-08-29T09:12:33.001Z"
}
authentication *
string (header)
An authorization header
psk-key *
string (header)
A pre-shared key for authorization

Responses

Code Description
201 Organization created successfully.
500 Server error.
